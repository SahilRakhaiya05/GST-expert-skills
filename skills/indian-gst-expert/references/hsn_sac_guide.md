## 1. What Are HSN/SAC Codes?

- **HSN codes** classify goods using an internationally standardized system (World Customs Organization). India uses 8-digit HSN codes for customs; for GST invoices, 4 or 6 digits are required.
- **SAC codes** are India-specific codes for classifying services under GST. All SAC codes start with **99**.
- The correct code determines the GST rate; wrong codes can lead to wrong tax charges, ITC denial, and penalties.

---

## 2. Mandatory Digit Requirements

| Annual Aggregate Turnover (AATO) | HSN Digits Required on Invoice | SAC Required? |
|---|---|---|
| Up to ₹5 Crore | **4-digit HSN** | Yes (for services) |
| Above ₹5 Crore | **6-digit HSN** | Yes (for services) |
| Export invoices | **8-digit HSN** | Yes |
| Import (customs) | **8-digit HSN** | Yes |

> **Note:** Composition scheme dealers with AATO ≤ ₹1.5 Crore are exempt from HSN on invoices, but must mention it in GSTR-4.

## 6. How to Look Up an HSN Code

1. **Official GST Portal:** gst.gov.in → Services → User Services → Search HSN/SAC Code
2. **ClearTax HSN Lookup:** cleartax.in/s/gst-hsn-lookup
3. **Structure of HSN codes:**
   - 2 digits = Chapter (broad product group, e.g. Chapter 85 = electrical machinery)
   - 4 digits = Heading
   - 6 digits = Sub-heading
   - 8 digits = Tariff item (for customs/export)

4. **Common Lookup Tip:** Use the first 2 digits to find the chapter, then narrow down.

| 06–14 | Vegetable products |
